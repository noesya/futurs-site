FUTURs est un jeu de société modulable mêlant bonne humeur, inspiration et action. 

Le jeu réconcilie les grands thèmes de société et les citoyens avec une approche ludique portée vers un futur heureux, désirable et prospère.

image: id: "1b561054-ebad-41a4-8c66-fd2dd8e6705e" file: "1b561054-ebad-41a4-8c66-fd2dd8e6705e" alt: >- credit: >- buttons: - title: >- Je commande le jeu url: >- https://www.ulule.com/futurs-le-jeu/ target_blank: true - kind: heading title: >- Rebattons les cartes position: 0 level: 2 - kind: block template: chapter title: >- position: 0 data: layout: no_background text: >-

Les 128 cartes mettent en dynamique le climat, la biodiversité, la justice sociale et le bien-être de manière inspirante. Plus de 5 modes jeu sont à votre disposition pour mettre en action FUTURs.

Vous trouverez un inventaire interactif des cartes référencées sur le site avec des explications ainsi que des ressources complémentaires (documentaires, podcasts, articles…). 

Vous pourrez aussi utiliser les cartes pour animer des ateliers établissements scolaires, le milieu associatif et les entreprises.

notes: >- image: id: "d54398f3-fc71-4972-844c-cfd8ac7101aa" file: "d54398f3-fc71-4972-844c-cfd8ac7101aa" alt: >- credit: >- - kind: heading title: >- Une initiative engagée position: 2 level: 2 - kind: block template: chapter title: >- position: 0 data: layout: no_background text: >-

FUTURs est un jeu au service des citoyens et de notre futur commun. De l’imagination du jeu à sa production, nous nous sommes engagés afin de faire de cette initiative un exemple. Nous avons choisi de travailler main dans la main avec Ludotopia pour que le jeu soit produit en France par des personnes en situation de handicap et avec des matériaux sains pour l’environnement.
