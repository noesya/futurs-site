FUTURs est un jeu modulable mêlant bonne humeur, inspiration et action. 

Le jeu réconcilie les grands thèmes de société et les citoyens avec une approche ludique portée vers un futur heureux, désirable et prospère.

image: id: "1b561054-ebad-41a4-8c66-fd2dd8e6705e" file: "1b561054-ebad-41a4-8c66-fd2dd8e6705e" alt: >- credit: >- buttons: - title: >- Je commande le jeu url: >- https://www.ulule.com/futurs-le-jeu/ target_blank: true - kind: heading title: >- 128 cartes pour imaginer demain position: 0 level: 2 - kind: block template: chapter title: >- position: 0 data: layout: no_background text: >-

FUTURs c'est avant tout un jeu de cartes mélant bonne humeur, enthousiasme, et espoir. Un jeu haut en couleurs pour faire naître de nouveaux souvenirs et partager des moments mémorables.

Les cartes du jeu s'organisent en plusieurs catégories représentant la cartographie de notre monde tel qu'il est aujourd'hui mais aussi tel qu'il pourraît advenir.

Serez-vous en mesure de rebattre les cartes pour un avenir prospère ?

Le jeu de société FUTURs est proposé en avant première avec une édition limitée disponible seulement lors de la campagne de financement qui est en cours.

Ne manquez pas l'opportunité de commander votre jeu afin de le recevoir avant Noël.

image: id: "1b561054-ebad-41a4-8c66-fd2dd8e6705e" file: "1b561054-ebad-41a4-8c66-fd2dd8e6705e" alt: >- credit: >- buttons: - title: >- Je commande le jeu url: >- https://www.ulule.com/futurs-le-jeu/ target_blank: true - kind: heading title: >- Une initiative engagée position: 1 level: 2 - kind: block template: chapter title: >- position: 0 data: layout: no_background text: >-

FUTURs est un jeu au service des citoyens et de notre futur commun. De l’imagination du jeu à sa production, nous nous sommes engagés afin de faire de cette initiative un exemple. 

Nous avons choisi de travailler main dans la main avec Ludotopia pour que le jeu soit produit en France par des personnes en situation de handicap et avec des matériaux sains pour l’environnement.
