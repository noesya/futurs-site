FUTURs est un jeu modulable mêlant bonne humeur, inspiration et action. 

Il réconcilie les grands thèmes de société et les citoyens avec une approche ludique portée vers un futur heureux, désirable et prospère.

alt: >- credit: >- buttons: - title: >- Je commande le jeu url: >- https://www.ulule.com/futurs-le-jeu/ target_blank: true - kind: heading title: >- 128 cartes pour imaginer un nouveau monde position: 0 level: 2 - kind: block template: chapter title: >- position: 0 data: layout: no_background text: >-

Découvrez un jeu haut en couleurs pour faire naître de nouveaux souvenirs et partager des moments mémorables.

Les cartes s'organisent en plusieurs catégories représentant la cartographie de notre monde tel qu'il est aujourd'hui et tel qu'il pourraît advenir.

FUTURs est un support multifonction : jeu en main, vous aurez le pouvoir de rebattre les cartes.
